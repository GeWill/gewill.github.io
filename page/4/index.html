7.3 类和结构体

类和结构体主要区别：

  1. 结构体不支持继承
  2. 结构体不支持定义析构器
  3. 结构体是值类型，而类是引用类型

引用类型的比较：

  • ===
  • !==

结构体是值类型的引用，不适用此运算符。

self 关键字：

  • 构造器中 self 代表该构造器正在进行初始化的实例
  • 方法中 self 代表该方法的调用者

类和结构体的选择

  • 结构体的主要目的是用于封装少量相关的简单数据值
  • 程序需要让该实例在赋值或参数传递时会自动复制副本
  • 程序明确该类型无需继承

7.5 计算属性

计算属性相当于 Objective-C、Java 等语言通过 getter、setter 合成的属性。

set 部分的形参可以省略，Swift 提供一个隐式的形参名：newValue

只有 get，没有 set 的计算属性称之为只读的计算属性。
